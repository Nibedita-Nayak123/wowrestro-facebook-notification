== Description ==
"Set Up a Facebook App: You need to create a Facebook App if you don't have one already. Go to the Facebook for Developers website, create a new app, and configure it.Get Page Access Token: To send messages on behalf of a Facebook Page (which is typically used for businesses), you need a Page Access Token. You can get this token by linking your app to your Facebook Page. Follow the Facebook documentation for details on how to do this.When your application receives the new order trigger, it can use the Page Access Token to send a notification message to the admin's Facebook user ID. You'll need to use the Facebook Graph API to send messages.."
== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "Wowrestro - Facebook Notifications".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see Wowrestro on the left navigation bar of WordPress Dashboard.
